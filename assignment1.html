A1: Laser Cut Press Fit Construction Kit!





Project Development Process

Write up of process

  • Created a new file in Adobe Illustrator, set the artboard size to match laser bed
  • Drew a 2.5 inch octagon
  • Measured cardboard thickness w/caliper = 3mm
  • Drew 16, 3mm wide "slot" rectangles
  • Rotated the rectangles to a 90 degree orientation to the octagon faces
  • Used the Pathfinder pallet to subtract the rectangles from the octagon
  • Set the line width to .001-in.

    Problems in file creation and how I resolved them

    • Initially I had difficulty in aligning my "slot" rectangles to exactly 90 degrees to the octagon edges
    • A Google search revealed that holding down the Shift key while rotating constrains the rotation angle to 45 degrees. With that I was able to rotate the rectangles to a perfect 90 degree orientation to the octagon edges
    • I also had some difficulty subtracting the "slot" rectangles from the octagon, but solved it moving the rectangles to the front using the "send to front" command, and the Pathfinder tool to subtract the rectangles from the octagon

    Kit design and execution

    • The shape of the octagon appealed to my tastes as it afforded the possibility of creating angled design patterns
    • My initial test fit was a success, which was a surprise. I learned to size the connectors to the exact size of the caliper measurement for best results
    • When it was my turn at the laser cutter, it was already properly set up. If I was to do this from the start, I would:
    • Ensure the laser cutter is turned on
    • Ensure the exhaust is turned on
    • Ensure air the compressor is enabled

    Setting the Focus over a workpiece

    • Loaded the aprx. 18" x 24" scrap cardboard I brought to class
    • Installed the focus key
    • Raised the z axis table height until light contact was with the cardboard
    • Removed the key

    File prep and export

    • Set laser printing settings to the following:
    • Speed = 25
    • Power = 100
    • Frequency = 100

    Final construction

    • Initial test fit was a success
    • Cut 21 instances of the slotted octagon
    • Constructed 8 different arrangements
    • For fun, made a post-in note holder
