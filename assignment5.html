A5: Subtractive Fabrication and CNC milling





Write up of process: Flat design and milling process

  • Decided to use finger joint method to make my box
  • Drew my box design in Adobe Illustrator
  • Exported the design as .DXF file
  • Imported the .DXF file into IKad
  • Saved a .grb file
  • Opened the file in Bantam Tools
  • Set the settings to (see image_)
  • Milled the file - success!
  • Pressfit the box

Problems in project and how I resolved them

  • The first try would not quite fit together
  • F1 has zero flexibilty, file must be perfect
  • It took several tries to get a perfect fit
  • Was close to my last piece of F1
  • With no F1 material left, could not iterate any further
  • Ultimately I had a result that was close to what I wanted

Bantam Tools Milling Machine settings

  • 1/32" bit was already installed when I arrived
  • Rehomed the bit
  • Made and F1 sandwich using double stick tape
  • Fixed the F1 sanwich to the base plate with double stick tape
  • Started the mill from Bantam Tools
  • After milling, vacuumed the particulates
